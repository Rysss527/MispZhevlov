# Описание игры

## О проекте

**Defense Synthesis** — это игра в жанре Tower Defense, разработанная на C++ с использованием библиотеки SFML. Проект вдохновлен классической игрой "Plants vs. Zombies" и включает в себя богатый набор игровых механик, таких как разнообразные типы башен и врагов, управление ресурсами, различные игровые состояния и продвинутую систему спецэффектов.
